Menu

Clique no menu ao lado para ver a manipulação de classes usando [class] e [ngClass].

Note que no primeiro comando, a classe is-active é adicionado ao elemento quando o condicional é verdadeiro. Quando o mesmo é falso, não é adicionado a classe is-active.

Já para o uso do [ngClass], foi usado a construção de if/else do tipo "condicional ? executa_se_verdaeiro : executa_se_falso". Na prática, o resultado é o mesmo

Note ainda que o [class.XXXX] pode ser usado quantas vezes quiser em um mesmo elemento. Por exemplo, no botão abaixo, a cor será determinada pelo elemento selecionado do menu:

  • Se não selecionou nada? Vermelho
  • Se selecionou um elemento impar? Vermelho
  • Se selecionou um elemento par? Vermelho
